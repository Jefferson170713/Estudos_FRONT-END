Editor de Plugins

Apenas pedindo para baixar o Live server, Prettier.

Live server é um servidor local que atualiza a página automaticamente.

Prettier é um formatador de código.

Color Highlight é um plugin que destaca as cores no código.

Bracket Pair Colorizer é um plugin que destaca os pares de chaves.

Material Icon Theme é um plugin que muda os ícones dos arquivos.

Auto Rename Tag é um plugin que renomeia a tag de fechamento automaticamente.
