Editor Atalhos

Usando alguns atalhos mais comuns no Vscode.

Ctrl + Shift + P é o atalho para abrir o painel de comandos.

Ctrl + P é o atalho para abrir o painel de arquivos.

Ctrl + Shift + E é o atalho para abrir o painel de explorador.

Alt + Seta pra cime ou Seta para baixo é o atalho para mover a linha.
