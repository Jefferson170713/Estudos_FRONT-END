CSS Básico

A ideia da aula é mostrar como funciona o CSS. O css é uma linguagem de estilização de páginas web.
Estilização em cascata, ou seja o estilo de um elemento pode ser sobrescrito por outro.
O css é composto por um seletor e uma declaração.

No arquivo css que escrevemos, tem o seletor a que tem a declaração color: red, text-decoration: none.
O atributo declarado text-decoration: none é para sumir a linha do link.
