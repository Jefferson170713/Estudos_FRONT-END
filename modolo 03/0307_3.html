0307 Flexbox 3

Use ambos no seu layout, o `flex` e o `grid`, para organizar os elementos de forma flexível e responsiva.
Eles reolvem problemas diferentes. O grid é ótimo para layouts de grade, enquanto o flex é ideal para alinhar itens em linha ou coluna.

Exemplo demonstra a diferença entre Flexbox e Grid para organização de elementos na tela:

  • `.flex` utiliza `display: flex`, permitindo alinhar itens em linha, com quebra automática para a próxima linha (`flex-wrap: wrap`) e espaçamento entre eles (`gap: 20px`). É ideal para distribuir elementos de forma flexível em uma direção.
  • `.grid` utiliza `display: grid` e `grid-template-columns: repeat(auto-fit, 300px)`, criando uma grade onde cada coluna tem 300px e o número de colunas se ajusta automaticamente ao tamanho da tela. É ótimo para criar layouts em grade, onde os itens se reorganizam conforme o espaço disponível.
