0304 Grid Column 1

Usando somente a propriedade display: grid todos os itens se comportam como se fossem colunas, ou seja, cada item ocupa uma coluna toda.
Com gap: 10px; o espaçamento entre os itens é de 10px.
Com o grid-template-columns: 1fr 1fr; o grid se comporta como duas colunas, ou seja, cada item ocupa 50% do espaço. E eu posso mudar a posição dos itens usando a propriedade grid-column: 1/2; ou seja, o item ocupa a coluna 1 e 2.
